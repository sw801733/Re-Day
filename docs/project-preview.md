# Re:Day

## 프로젝트 목적

Re:Day는 Notion에 작성한 일기를 기반으로  
주 2회(월/목 오전 7:30)에 Slack DM으로 리마인드 카드를 보내주는 개인 회고 자동화 도구다.

목표는 행동 유도나 자기계발 조언이 아니라  
최근의 상태와 흐름을 다시 마주치게 하는 것이다.

---

# MVP 기능

이번 MVP에서는 아래 기능만 구현한다.

1. Notion Daily_Log DB에서 최근 일기 조회
2. 직전 전송 이후 새 일기 존재 여부 확인
3. 새 일기가 있을 때만 요약 생성
4. Slack DM 전송
5. GitHub Actions로 월/목 07:30 자동 실행

---

# 전송 규칙

## 실행 시점
- 월요일
- 목요일
- 오전 07:30 (KST)

## 전송 조건
- 마지막 전송 이후 새로운 일기가 있을 때만 전송
- 새 일기가 없으면 아무 것도 보내지 않음

# 리마인드 카드 방향

### 말투
- 반말

### 스타일
- 담백함
- 상태 중심
- 감정 흐름 중심

### 금지 요소
- 조언
- 자기계발식 문장
- 행동 지시
- 교훈 정리

예:
❌ "이번 주는 조금 더 쉬어보는 것도 좋겠다"  
❌ "이제 루틴을 만들면 좋겠다"

# 시스템 흐름

1. 스케줄 실행 (GitHub Actions)
2. 상태 파일 읽기
3. Notion DB에서 최근 일기 조회
4. 새 일기 필터링
5. 새 기록 없으면 종료
6. OpenAI 요약 생성
7. Slack DM 전송
8. 상태 업데이트

# 성공 기준

아래가 되면 MVP 성공이다.

- 월/목 07:30 자동 실행
- 새 일기 없으면 메시지 없음
- 새 일기 있으면 Slack DM 전송
- 중복 전송 없음
